# AtividadeCSS

Nome: Pedro Daniel Souza Leal

O objetivo desta atividade é que você ponha em prática o que você pode aprender de HTML e CSS até aqui.  Você irá criar um site com no mínimo 4 páginas, sobre o tema que você quiser, utilizando HTML e CSS. O conteúdo pode ser fictício, use a sua criatividade.

Requisitos

- Estrutura HTML válida em todas as páginas
- Cabeçalho padronizado com logo, título e menu de navegação
- Rodapé padronizado com informações de contato
- Elementos de texto, imagens e vídeo do YouTube
- Lista com subitens
- Formulário com no mínimo 5 controles
- Conteúdo centralizado horizontalmente com margens
- Uso de pseudo-classes
- Layout organizado com Flexbox
- Mínimo de 3 cores e 3 fontes diferentes
- Conteúdos organizados lado a lado
